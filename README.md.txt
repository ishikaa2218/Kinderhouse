KinderHouse is a responsive eCommerce toy store built using React, focused on performance, usability, and clean user experience. The project demonstrates practical frontend development skills through dynamic product handling, filtering, and cart management in a real-world shopping flow.

Features

* Responsive and mobile-friendly UI
* Dynamic product listing
* Category-based and price-based filtering
* Add to cart and remove from cart functionality
* Cart item quantity management
* Reusable React components
* Clean and intuitive user interface
* Optimized rendering for better performance

Tech Stack

Frontend

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* Bootstrap

Tools & Practices

* Component-based architecture
* State management using React hooks
* Modular and maintainable code structure
* Git & GitHub for version control

* Handling user interactions in eCommerce flows
